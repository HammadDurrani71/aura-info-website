# AURA — Intelligent Analytics & Security Dashboard

> Your security, reinvented.

AURA is an AI-driven video analytics platform that transforms standard CCTV systems into intelligent security tools with cross-camera person tracking, natural language search, real-time alerts, and interactive floor map visualization.

Built by **Team VisionLink** as a Capstone Project at the University of Wollongong Dubai (UOWD).
